# Sample Skill

このスキルは、SKILLの最小構成を確認したいときの雛形です。

## 使う場面

- 新しいSKILLを作る前に書式を確認したいとき
- 手順書を短く構造化して渡したいとき

## ワークフロー

1. 要件を1行で要約する。
2. 成果物の保存先を明示する。
3. 手順を3〜5ステップで書く。
4. 検証方法を1つ添える。

## 出力テンプレート

```md
# <Task Name>

## Goal
<1行の目的>

## Steps
1. <step1>
2. <step2>
3. <step3>

## Validation
- <確認コマンド/確認観点>
```
